Swap Manager: USD vs SORF Swap Valuation

A powerful tool designed for pricing USD vs SOFR interest rate swaps, empowering users to customize Spot Rates and swap cashflow details effortlessly. The calculator swiftly computes Discount Factors, Reset Rates, and Present Values for each leg, providing a comprehensive view of the swap. With the added convenience of downloading swap curves and associated discount factors directly to an Excel file, financial analysis becomes seamless.

This project was meticulously crafted using Python, JavaScript, the Django Framework, and the HTMX library.
